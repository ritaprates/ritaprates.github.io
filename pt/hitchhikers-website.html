The Hitchhiker's Guide to the Galaxy website responsive

Este site é resultado de um projecto da cadeira de Webdesign, em que os alunos tinham que criar um site para um filme que não tivesse um; eu escolhi The Hitchhiker's Guide to the Galaxy. Durante o segundo semestre tivemos que transformar o nosso site existente num site responsive, pelo que este é o resultado final. É possível visitar o site em hitchhikersguidegalaxy.github.io, mas os tempos de carregamento são um pouco lentos, uma vez que, por ser um trabalho universitário, foi usado um serviço de alojamento gratuito. Esta foi a minha experiência com HTML e CSS e é interessante ver a minha evolução desde que fiz o design e código para este site.
Janeiro/Junho 2015
